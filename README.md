Photometry Pipeline.ipynb : jupyter notebook containing code to run the photometry pipeline on raw BVR images and generate a catalog of magnitudes.
sm_phot.py : custom package written to support the data pipeline with functions for image processing.
BVR Photometry of NGC-7789 : report on the results of this study. Relevant information to the functions of this repository can be found in section 3.1.

Instructions:
Download Photometry Pipeline.ipynb and sm_phot.py to the same folder.
Organize science and calibration images and modify filepaths in notebook.
Run full notebook. Resulting data product is a catalog of magnitudes for each filter for all sufficiently bright targets in view.
